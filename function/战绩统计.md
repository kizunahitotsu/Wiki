# 战绩统计
显示卡片对战胜率统计（当前所在段位）的页面。
### 全站战绩统计
注意：10月1日清空历史战绩，重新开始统计胜率表。   
表格查看方式为“纵列红色卡片”对“横行蓝色卡片”的胜率，区分进攻和防守。   
胜率大于90%显示为深绿色、胜率大于70%显示为浅绿色、胜率小于30%显示为浅红色。空白表示对应战斗次数过少没有统计价值。
